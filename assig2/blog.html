Entry 6: Concept about Programming and JavaScript

Programming Language is a way to describe to a computer some behaviour. For instance a calculation or another action that you want the computer to do for you(Therefore a programming language describes a program). Javascript is a programming language which lets you access and modify the content and markup of webpages while the user is viewing/interacting with them and it can respond to user events(entering some text) as well as combining them together to form a logical espressed in programs.

An example would be adding two variables together. In this example when a user clicks this paragraph, a function is called or executed and will sum two variables together stored in a new variable and will display this variables result being 15 in an alert message.
