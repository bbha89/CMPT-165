HTML
HTML also called the HyperText Markup Language is used to describe the content of pages.
Markup Language
A markup language is a way to describe a document for a computer using only plain text.
Semantic Markup
HTML elements (and some attributes) that are not intended to affect the structure of your webpages, but they add extra information to the pages and describe contents more accurately.
